# Installing into Slack

- clone this repo
```
git clone git@github.com:GetGoodGames/slack-theme.git
```

- run the setup command for your operating system, you may have to `sudo`
```
cd slack-theme
sudo ./setup.sh # linux
sudo ./setup.command # OS X
```

- completely restart Slack
- Enjoy!
